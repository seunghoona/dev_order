# 🔷 사용자 스토리

## 회원 
- 사용자는 회원가입을 진행한다.
- 사용자는 로그인시 토큰 값을 얻는다.

### 회원 필수 값 
- email, password 필수이다.
- 유저의 권한

---

## 상품주문
- 사용자가 로그인한 경우 상품 주문 가능

### 상품주문 필수 값
- 고유한 주문번호
- 상품을 주문한 회원 키
- 주문한 상품
- 주문한 상태

---
## 회원 주문 내역
- 상품의 주문시 해당 사용자의 상품 내역을 저장한다.

### 회원 주문내역 필수 값
- 주문상품번호 (주문일(8)+ KM(크몽) + 상품번호 + 회원번호)
- 주문상품_ID
- 주문 상태 
- 주문생성일자
