

## 요구사항 

|용어|영문명|의미|
|:---:|:---:|:---:|
|회원  |member |사용자는 로그인 및 회원가입|
|상품  |product |회원이 주문하고자하는 상품 등록|
|상문주문|orders|사용자는 상품을 주문|
|상품내역|ordersHistory|회원이 주문한 상품 내역을 확인|


- [x] 회원가입
  - id 회원별 고유한 아이디 
  - email 회원별 고유한 email 
  - password
- [x] 로그인
- [ ] 로그아웃
- [x] 상품
  - 상품명, 상품금액
- [x] 상품조회
- [ ] 상품주문
- [ ] 회원주문내역조회 

## 회원
- 사용자는 회원가입을 진행한다. 
  - email, password 필수이다. 
  - 사용자의 권한

## 상품주문
- 고유한 주문번호
- 상품을 주문한 회원 키
- 주문한 상품
- 주문한 상태

## 회원 주문
- 상품의 주문시 내역을 저장한다.
